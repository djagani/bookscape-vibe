# Audio Files for Environments

Place your ambient audio files in this directory for each genre environment.

## Required Files:

1. **romance.mp3** - Soft, romantic music (piano, strings, gentle melodies)
2. **fantasy.mp3** - Mystical, magical music (ethereal sounds, harps, flutes)
3. **thriller-mystery.mp3** - Tense, suspenseful music (low drones, ambient tension)
4. **dystopian.mp3** - Industrial, gritty sounds (electronic, mechanical)
5. **historical-mythological.mp3** - Classical, ancient music (traditional instruments, orchestral)
6. **melancholic.mp3** - Sad, reflective music (slow piano, cello, rain sounds)

## Where to Find Royalty-Free Audio:

- **Pixabay**: https://pixabay.com/music/
- **Free Music Archive**: https://freemusicarchive.org/
- **Incompetech**: https://incompetech.com/music/
- **YouTube Audio Library**: Free music for creators

## Tips:

- Keep files under 5MB for faster loading
- Use .mp3 format for best browser compatibility
- Volume is set to 30% by default in the app
- Audio loops automatically

## Alternative:

If you don't have audio files, the environments will still work with visual effects only. The audio will simply not play.
